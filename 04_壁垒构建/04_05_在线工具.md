# 在线Skills/Plugin工具

> **模块目标**: 掌握主流在线AI开发平台，无需本地部署也能创建强大的AI应用

---

## 一、在线工具概述

### 1.1 为什么需要在线工具

在前面两章，我们学习了如何本地开发和部署Skills、Plugin。但本地开发有一些限制：

| 限制 | 说明 |
|-----|------|
| **技术门槛** | 需要一定的编程基础 |
| **环境配置** | 需要配置开发环境 |
| **分享困难** | 分享给他人使用较麻烦 |
| **维护成本** | 需要自己维护服务器 |

**在线工具的优势**：

| 优势 | 说明 |
|-----|------|
| **零代码/低代码** | 可视化操作，降低技术门槛 |
| **即开即用** | 无需配置环境 |
| **一键分享** | 轻松分享给他人使用 |
| **平台维护** | 平台负责服务器和更新 |

### 1.2 主流在线工具

| 工具名称 | 开发者 | 特点 | 适用场景 |
|---------|--------|------|---------|
| **Coze** | 字节跳动 | 易上手、AI能力强、免费 | AI Bot开发、智能问答 |
| **Dify** | 开源 | 可私有部署、功能全面 | 企业AI应用、知识库 |
| **FastGPT** | 开源 | 知识库问答、易部署 | 知识库应用 |
| **GPTs** | OpenAI | 与ChatGPT集成 | ChatGPT用户 |
| **扣子** | 字节跳动 | Coze的国内版 | 国内用户 |

## 二、Coze平台深度使用

### 2.1 Coze平台介绍

Coze是字节跳动推出的AI应用开发平台，让你能够快速创建、调试和部署AI Bot。

**核心特点**：

| 特点 | 说明 |
|-----|------|
| **免费使用** | 基础功能完全免费 |
| **可视化编辑** | 拖拽式操作，无需编程 |
| **丰富的插件** | 内置大量现成插件 |
| **多渠道发布** | 可发布到飞书、微信等 |
| **工作流支持** | 支持复杂的工作流编排 |

**Coze vs 本地开发**：

| 对比维度 | 本地开发 | Coze平台 |
|---------|---------|---------|
| 技术门槛 | 需要编程 | 可视化操作 |
| 开发速度 | 较慢 | 快速 |
| 灵活性 | 高 | 中等 |
| 部署难度 | 需要服务器 | 一键发布 |
| 成本 | 服务器费用 | 免费基础版 |

### 2.2 创建Bot

#### 步骤1：注册账号

1. 访问 https://www.coze.com/（国际版）或 https://www.coze.cn/（国内版）
2. 使用邮箱或手机号注册
3. 完成账号设置

#### 步骤2：创建新Bot

1. 点击"Create Bot"
2. 填写基本信息：
   - **名称**：给Bot起一个名字
   - **描述**：简短描述Bot的功能
   - **图标**：上传或生成一个图标

#### 步骤3：设计人设

人设（Persona）决定了Bot的性格和行为方式。

**人设编写模板**：

```markdown
## 角色设定
你是一个[角色描述]，专门帮助用户[核心功能]。

## 能力范围
- [能力1]
- [能力2]
- [能力3]

## 回答风格
- [风格特点1]
- [风格特点2]

## 限制
- 不回答[某类问题]
- 不执行[某类操作]
```

**示例：周报助手Bot人设**

```markdown
## 角色设定
你是一个专业的周报撰写助手，帮助用户快速生成规范的工作周报。

## 能力范围
- 根据用户提供的信息生成周报
- 优化和润色现有周报
- 提供周报写作建议

## 回答风格
- 专业、简洁
- 结构清晰
- 突出重点

## 限制
- 不替用户编造工作内容
- 不提供虚假数据
```

### 2.3 添加Skills

Coze中的Skills称为"技能"或"插件"。

#### 使用内置插件

1. 在Bot编辑页面，点击"Plugins"
2. 浏览插件市场
3. 点击"+"添加需要的插件

**常用内置插件**：

| 插件名称 | 功能 |
|---------|------|
| **搜索** | 联网搜索信息 |
| **天气** | 查询天气 |
| **新闻** | 获取新闻资讯 |
| **计算器** | 数学计算 |
| **翻译** | 多语言翻译 |

#### 创建自定义技能

如果内置插件不满足需求，可以创建自定义技能：

1. 点击"Skills" → "Create Skill"
2. 编写技能描述和提示词
3. 配置输入参数
4. 测试并保存

**自定义技能示例**：

```
技能名称：周报生成器

描述：根据用户输入生成工作周报

提示词：
你是一个周报撰写专家。请根据以下信息生成周报：

本周完成：
{{completed}}

进行中：
{{ongoing}}

问题：
{{problems}}

下周计划：
{{plans}}

要求：
1. 结构清晰
2. 语言专业
3. 突出成果
4. 控制在500字内
```

### 2.4 添加Plugin

Coze支持添加外部API作为Plugin。

#### 配置API Plugin

1. 点击"Plugins" → "Add Plugin"
2. 选择"Create New"
3. 配置API信息：
   - **API URL**：接口地址
   - **Method**：请求方法（GET/POST）
   - **Headers**：请求头
   - **Parameters**：参数定义

**示例：添加天气API**

```
名称：天气查询
描述：查询指定城市的天气

API配置：
- URL: https://api.weather.com/v1/current
- Method: GET
- Parameters:
  - city (string, required): 城市名称

响应处理：
从返回的JSON中提取天气信息并格式化输出
```

### 2.5 工作流编排

Coze支持使用工作流来编排复杂的任务。

#### 工作流基本概念

| 概念 | 说明 |
|-----|------|
| **触发器** | 启动工作流的条件 |
| **节点** | 执行具体任务的单元 |
| **连线** | 节点之间的执行顺序 |
| **变量** | 在节点间传递的数据 |

#### 创建简单工作流

**示例：智能日报生成工作流**

```
[触发器] → [获取日期] → [调用日历API] → [AI整理] → [格式化输出]
```

**步骤**：

1. 创建新工作流
2. 添加"开始"节点
3. 添加"日期处理"节点
4. 添加"LLM"节点（AI处理）
5. 添加"结束"节点
6. 连接各节点
7. 配置变量传递
8. 测试并保存

### 2.6 发布与分享

#### 发布渠道

| 渠道 | 说明 |
|-----|------|
| **Coze商店** | 发布到Coze官方商店 |
| **飞书** | 集成到飞书机器人 |
| **微信** | 集成到微信公众号 |
| **API** | 通过API调用 |
| **Web** | 生成独立的网页 |

#### 发布步骤

1. 完成Bot开发和测试
2. 点击"Publish"
3. 选择发布渠道
4. 填写发布信息
5. 提交审核（如需要）
6. 发布成功

#### 使用统计

发布后可以查看：
- 使用人数
- 对话次数
- 用户反馈
- 错误日志

## 三、Dify入门

### 3.1 Dify的特点

Dify是一个开源的LLM应用开发平台，可以私有部署。

| 特点 | 说明 |
|-----|------|
| **开源免费** | 完全开源，可免费使用 |
| **私有部署** | 数据完全在自己手中 |
| **功能全面** | 支持多种AI应用类型 |
| **可视化开发** | 低代码开发方式 |

### 3.2 创建AI应用

Dify支持多种应用类型：

| 应用类型 | 说明 | 适用场景 |
|---------|------|---------|
| **聊天助手** | 对话式AI | 客服、问答 |
| **文本生成** | 生成式AI | 写作、翻译 |
| **Agent** | 智能代理 | 复杂任务 |
| **工作流** | 流程编排 | 多步骤任务 |

#### 创建步骤

1. 登录Dify平台
2. 点击"创建应用"
3. 选择应用类型
4. 配置模型和提示词
5. 添加工具和知识库
6. 测试并发布

### 3.3 知识库集成

Dify的知识库功能非常强大：

**支持的文档类型**：
- TXT、Markdown
- PDF、Word
- Excel、CSV
- 网页链接

**使用步骤**：

1. 创建知识库
2. 上传文档
3. 等待索引完成
4. 在应用中关联知识库
5. 测试问答效果

### 3.4 Dify vs Coze

| 对比维度 | Coze | Dify |
|---------|------|------|
| **部署方式** | 云端 | 云端/私有 |
| **费用** | 免费基础版 | 开源免费 |
| **数据隐私** | 存储在云端 | 可完全私有 |
| **易用性** | 更简单 | 稍复杂 |
| **功能深度** | 中等 | 更深 |

**选择建议**：
- 🎯 个人用户、快速开发：选择Coze
- 🏢 企业用户、数据敏感：选择Dify私有部署

## 四、工具选择建议

### 4.1 场景匹配

| 场景 | 推荐工具 | 理由 |
|-----|---------|------|
| **快速创建AI Bot** | Coze | 操作简单，上手快 |
| **企业内部应用** | Dify | 可私有部署，数据安全 |
| **知识库问答** | Dify/FastGPT | 知识库功能强大 |
| **与ChatGPT集成** | GPTs | 无缝集成 |
| **国内用户使用** | Coze国内版 | 访问稳定 |

### 4.2 成本考量

| 工具 | 免费额度 | 付费方案 |
|-----|---------|---------|
| **Coze** | 基础功能免费 | 高级功能付费 |
| **Dify云版** | 有限免费 | 按使用量付费 |
| **Dify私有** | 完全免费 | 服务器成本 |
| **GPTs** | 需要ChatGPT Plus | $20/月 |

### 4.3 团队协作

| 需求 | 推荐方案 |
|-----|---------|
| **多人协作开发** | Coze团队版 / Dify |
| **权限管理** | Dify |
| **版本控制** | Dify + Git |
| **共享给团队使用** | Coze发布 / Dify API |

## 五、实战案例

### 5.1 创建一个智能客服Bot

- 自动回答常见问题
- 支持转人工
- 记录用户反馈

   - 人设：专业、友好的客服

   ```
   用户提问 → 知识库检索 → AI回答 → 满意度调查
   ```

   - 发布到飞书/微信

   ```markdown

   - 本周完成：{{completed}}
   - 进行中：{{ongoing}}
   - 问题：{{problems}}
   - 下周计划：{{plans}}
   ```

3. **添加模板选择**
