# 如何安装一个RPA

> **模块目标**: 成功安装并配置好一款RPA工具，为后续的自动化实践做好准备

---

## 一、安装前的准备

### 1.1 系统要求

大多数RPA工具对电脑配置有一定要求，以下是通用建议：

| 配置项 | 最低要求 | 推荐配置 |
|-------|---------|---------|
| **操作系统** | Windows 10 | Windows 10/11 64位 |
| **内存** | 4GB | 8GB以上 |
| **硬盘空间** | 2GB可用空间 | 10GB以上SSD |
| **处理器** | 双核CPU | 四核及以上 |
| **屏幕分辨率** | 1366×768 | 1920×1080及以上 |

> ⚠️ **注意**: 大多数RPA工具主要支持Windows系统。如果你使用Mac，可以考虑：
> - 使用虚拟机（如Parallels Desktop）
> - 使用云端RPA服务
> - 选择支持Mac的工具（如Keyboard Maestro）

### 1.2 账号准备

在安装前，建议先准备好：

1. **官网账号**: 大多数RPA工具需要注册账号
2. **邮箱**: 用于接收验证邮件和通知
3. **手机号**: 部分工具需要手机验证

### 1.3 工具选择确认

本章以**影刀RPA**为例进行讲解，原因如下：

-✅ 完全中文界面
- ✅ 免费版功能完整
- ✅ 安装简单
- ✅ 社区支持好

如果你选择其他工具，安装流程类似，可参考官方文档。

## 二、影刀RPA安装教程

### 2.1 下载安装包

#### 步骤1：访问官网

1. 打开浏览器，访问影刀官网：https://www.yingdao.com/
2. 点击页面上的"免费下载"或"立即体验"按钮

#### 步骤2：注册账号

1. 点击"注册"按钮
2. 输入手机号，获取验证码
3. 设置密码，完成注册

#### 步骤3：下载客户端

1. 登录后，进入"下载中心"
2. 选择适合你系统的版本（Windows 64位）
3. 点击下载，等待下载完成

### 2.2 安装步骤

#### 步骤1：运行安装程序

1. 找到下载的安装包（通常在"下载"文件夹）
2. 双击运行安装程序
3. 如果出现安全提示，点击"是"或"允许"

#### 步骤2：选择安装路径

```
默认路径：C:\Program Files\Yingdao\RPA
建议：保持默认，或安装到SSD硬盘
```

> 💡 **提示**: 避免安装到中文路径或包含特殊字符的路径

#### 步骤3：完成安装

1. 勾选"我同意用户协议"
2. 点击"立即安装"
3. 等待安装进度条完成
4. 勾选"立即启动影刀"
5. 点击"完成"

### 2.3 首次启动配置

#### 步骤1：登录账号

1. 启动影刀客户端
2. 输入注册时的手机号和密码
3. 点击"登录"

#### 步骤2：新手引导

首次登录会弹出新手引导：

1. **选择身份**: 选择"个人用户"或"企业用户"
2. **选择场景**: 勾选你感兴趣的自动化场景
3. **完成引导**: 点击"开始体验"

#### 步骤3：界面熟悉

登录后，你会看到影刀的主界面：

```
┌─────────────────────────────────────┐
│菜单栏: 文件 | 编辑 | 视图 | 帮助│
├──────────┬──────────────────────────┤
│          │                          │
│  组件库   │        流程编辑区        │
│          │                          │
│  - 操作类 │                          │
│  - 数据类 │                          │
│  - 逻辑类 │                          │
│          │                          │
├──────────┴──────────────────────────┤
│           控制台/日志区               │
└─────────────────────────────────────┘
```

## 四、安装后的基础配置

### 4.1 通用配置建议

无论使用哪款RPA工具，建议进行以下基础配置：

#### 1. 自动保存设置

```
路径：设置 → 通用 → 自动保存
建议：开启，间隔设为5分钟
```

#### 2. 日志级别设置

```
路径：设置 → 调试 → 日志级别
建议：开发阶段设为"详细"，生产阶段设为"错误"
```

#### 3. 快捷键设置

```
路径：设置 → 快捷键
建议：熟悉常用的快捷键，提升效率
```

### 4.2 影刀RPA推荐配置

#### 开启开发者模式

1. 点击右上角"设置"图标
2. 选择"高级设置"
3. 开启"开发者模式"
4. 重启客户端

#### 配置浏览器插件

影刀支持Chrome和Edge浏览器插件：

1. 点击"设置" → "浏览器插件"
2. 点击"安装Chrome插件"
3. 在Chrome扩展程序页面启用插件

## 五、常见安装问题排查

### 5.1 安装失败问题

#### 问题1：提示"缺少.NET Framework"

**原因**: 系统缺少必要的运行环境

**解决方案**:
1. 访问微软官网下载.NET Framework
2. 安装后重新运行RPA安装程序

#### 问题2：提示"权限不足"

**原因**: 当前账户没有管理员权限

**解决方案**:
1. 右键点击安装程序
2. 选择"以管理员身份运行"

#### 问题3：安装卡在某个进度

**原因**: 可能被杀毒软件拦截

**解决方案**:
1. 暂时关闭杀毒软件
2. 将RPA安装目录添加到白名单
3. 重新安装

### 5.2 启动问题

#### 问题1：登录后闪退

**可能原因**:
- 网络问题
- 账号异常
- 缓存冲突

**解决方案**:
1. 检查网络连接
2. 清除客户端缓存
3. 重新安装客户端

#### 问题2：提示"版本过低"

**解决方案**:
1. 卸载旧版本
2. 下载最新版本重新安装

### 5.3 运行问题

#### 问题1：流程运行无反应

**排查步骤**:
1. 检查是否有错误日志
2. 确认目标应用是否已打开
3. 检查屏幕分辨率是否匹配

#### 问题2：浏览器操作失败

**解决方案**:
1. 确认浏览器插件已安装并启用
2. 检查浏览器版本是否兼容
3. 尝试重新安装插件
